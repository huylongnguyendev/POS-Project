#Introduce

This mini project is a pos simulator,,that designed for fashion shop. It has two in one screen, which one is a main screen has product list, shopping cart with detail about price, quantity, total price. The other is an order list has detail for each order, update list in realtime instantly.

## Techs

- **Frontend**
  - Languages: HTML/CSS,TypeScript.
  - Framework/Libary: ReactJS, Tailwind.
  - Styling Component UI: Shadcn
  - Networking: Axios
  - State Management: Redux Toolkit

- **Backend**
  - Language: C#.
  - Framework: ASP.NET.

- **Others**
  - Mangae and Store source code: Git/Github.
